# Common Calculus Mistakes

## Limits
1. Applying L’Hôpital’s rule without indeterminate form
2. Ignoring one-sided limits
3. Treating ∞ − ∞ as 0

## Derivatives
1. Forgetting chain rule
2. Using (fg)' = f′g′
3. Sign errors in quotient rule
4. Missing implicit differentiation terms

## Integration
1. Forgetting constant of integration
2. Ignoring absolute value in ∫1/x dx
