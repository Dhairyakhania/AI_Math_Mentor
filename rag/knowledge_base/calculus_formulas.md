# Derivatives

## Basic Rules
- d/dx(c) = 0
- d/dx(xⁿ) = nxⁿ⁻¹
- d/dx(f + g) = f′ + g′
- d/dx(fg) = f′g + fg′
- d/dx(f/g) = (f′g − fg′)/g²
- Chain rule: d/dx[f(g(x))] = f′(g(x))·g′(x)

## Common Derivatives
- sin x → cos x
- cos x → −sin x
- tan x → sec²x
- eˣ → eˣ
- ln x → 1/x

# Limits

## Standard Limits
lim(x→0) sin x / x = 1  
lim(x→∞) (1 + 1/x)ˣ = e  

## L’Hôpital’s Rule
Applicable only for 0/0 or ∞/∞ forms
