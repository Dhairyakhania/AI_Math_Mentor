# Linear Algebra Basics

## Determinants
det(AB) = det(A)·det(B)  
det(A⁻¹) = 1/det(A)

## Matrix Inverse (2×2)
A⁻¹ = (1/det A) [ d  −b  
                   −c  a ]

## Transpose
(AB)ᵀ = BᵀAᵀ  
(A⁻¹)ᵀ = (Aᵀ)⁻¹

## Eigenvalues
det(A − λI) = 0  
Sum = trace(A)  
Product = det(A)

## Cramer’s Rule
xᵢ = det(Aᵢ) / det(A)
