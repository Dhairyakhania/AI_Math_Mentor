# Common Probability Mistakes

1. Confusing independence with mutual exclusivity
2. Mixing up P(A|B) and P(B|A)
3. Forgetting complementary probability
4. Probability value exceeding 1
5. Assuming independence without justification
6. Counting permutations when combinations are required
